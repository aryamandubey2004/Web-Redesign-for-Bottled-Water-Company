# Shree Krishna Corporation

A clean, responsive, single-page static website built for a Bisleri distributor.

## Overview

This website provides a modern, mobile-friendly design using popular front-end frameworks and libraries. The design ensures smooth navigation, fast loading, and easy accessibility for users.

## Tech Stack

- **HTML5 / CSS3 / JavaScript**
- **Bootstrap** — Responsive front-end framework
- **jQuery** — DOM manipulation and event handling
- **Modernizr** — Feature detection
- **Sticky.js** — Sticky navigation behavior
- **jQuery Easing** — Smooth scrolling animations
- **Bootsnav** — Multi-purpose navigation bar plugin
